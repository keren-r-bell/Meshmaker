# Meshmaker  

<img src="/Screenshot.png" alt="Screenshot of Meshmaker, a utility for making Mesh Gradients for SwiftUI projects."></img>

**Meshmaker** lets you create Mesh Gradient elements for your SwiftUI code in an easy to use, precise canvas.  
  
This app was made because of how frustrating I found coding even simple placeholder visuals using the very visually pleasing [MeshGradient](https://developer.apple.com/documentation/SwiftUI/MeshGradient) structure, so I figured someone should make a tool for it, and despite some options existing online, I still felt compelled to make my own. 

This aims to be a comprehensive canvas editing tool for its purpose, complete with modifier keys, keyboard shortcuts and color harmony.
